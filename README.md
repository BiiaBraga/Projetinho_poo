# Projetinho POO 🐾💻

Este projeto foi desenvolvido durante as aulas do **bootcamp de Desenvolvimento FullStack JavaScript da Generation Brasil** com o objetivo de praticar conceitos de **Programação Orientada a Objetos (POO)** utilizando **TypeScript**.

A aplicação simula um **sistema de cadastro de animais**, onde é possível trabalhar com diferentes tipos de animais, como **mamíferos** e **aves**.

O projeto foi desenvolvido em conjunto com o instrutor durante as aulas para reforçar a compreensão dos conceitos de orientação a objetos.

---

## 📚 Conceitos estudados

Durante o desenvolvimento deste projeto foram trabalhados os seguintes conceitos de POO:

- Classes
- Construtor
- Getters e Setters
- Classe Abstrata
- Herança
- Model
- Repository (Interface)
- Controller

---

## 🐾 Funcionalidade do projeto

O sistema permite realizar o **cadastro de animais**, diferenciando os tipos de animais utilizando conceitos de herança e abstração.

Exemplos de tipos de animais no sistema:

- Mamíferos
- Aves

## ⚙️ Pré-requisitos

Antes de executar o projeto, é necessário ter instalado em sua máquina:

- Node.js
- npm
- TypeScript
- ts-node

### Instalar Node.js

Baixe pelo site oficial:

https://nodejs.org

Após instalar, verifique no terminal:

```bash
node -v
npm -v
```

Se aparecerem as versões instaladas, significa que o Node.js e o npm estão funcionando corretamente.

---

### Instalar o TypeScript

No terminal, execute:

```bash
npm install -g typescript
```

Depois verifique a instalação:

```bash
tsc -v
```

---

### Instalar o ts-node

O **ts-node** permite executar arquivos TypeScript diretamente no terminal sem precisar compilá-los para JavaScript.

Instale com o comando:

```bash
npm install -g ts-node
```

## 📥 Como clonar o repositório

No terminal, execute:

```bash
git clone https://github.com/BiiaBraga/Projetinho_poo.git
cd Projetinho_poo
```

---

## 📦 Instalação das dependências

Após clonar o repositório, instale as dependências listadas no arquivo `package.json`:

```bash
npm install
```

Esse comando irá instalar automaticamente as bibliotecas utilizadas no projeto, como:

- readline-sync
- @types/node
- @types/readline-sync

---

## ▶️ Como executar os arquivos

Para executar o sistema utilize o comando:

```bash
ts-node Menu.ts
```

A aplicação será executada no terminal.
